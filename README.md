# Revolver
Revolver: Vetex-centric Graph Partitioning Using Reinforcement Learning

## Compile
~~~
    cd devel
    make clean && make all
~~~

## Run
~~~
    ./main -n <num_partitions> 
           -a <revolver|spinner|random|hash|range|pagerank>
           -f [<edge_file_name> <comment_style> <delimiter> <columns>]
~~~
